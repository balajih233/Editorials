Two Sum

The given problem says that there are n numbers present in an array and a target. We need to find two numbers which sums up target and return indexes of two numbers.


Let us take an example of an array arr[] = [4,3,9,6,2] and a target element 11.


From this array elements we can say that index at 2 and index at 4 sums up to 11. arr[2] + arr[4] = 11. So we need to return indexes 2 and 4.


If we need to return boolean value then we can do this with two pointers approach. But the given solution should return indexes. So I will take a Hash Map and store the indexes of each element. Keys of HashMap are elements and values are indexes.


I will iterate through array and check whether difference between array element and target is exit in map or not. If the difference is present then I will return current index and difference's index.


At index 0, element is 4 is present. I will calculate difference between element(4) and target(11) which is 7. Currently, My HashMap is empty so i will add element 4 and value as 0.

Time Complexity : O(n)

Space Complexity : O(n)

Java Code: class Solution { public int[] twoSum(int[] nums, int target) { int ans[]=new int[2]; HashMap<Integer,Integer> x=new HashMap<>(); for(int i=0;i<nums.length;i++){ int diff = target - nums[i]; if(x.containsKey(diff)){ ans[0] = x.get(diff); ans[1] = i; return ans; } x.put(nums[i],i); } return -1; } }
